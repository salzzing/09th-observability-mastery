# 박준우

## 역할

- 스터디장

## 관심 분야

- Cloud
- Kubernetes
- Observability
- SRE / DevOps
- Platform Engineering

## 이 스터디에서 얻고 싶은 것

- Observability 개념을 운영 관점에서 체계적으로 정리하기
- 장애 상황에서 Metrics, Logs, Traces를 어떻게 연결해서 볼지 훈련하기
- Prometheus, Grafana, OpenTelemetry 흐름을 스터디원들과 함께 학습하기
- 스터디 결과물을 GitHub에 아카이빙하기

## 개인 목표

- 매주 세션 요약 정리
- 참여자들이 질문과 발표를 편하게 할 수 있는 구조 만들기
- 후반부 실습을 Base 방향과 Project 방향으로 나누어 안정적으로 운영하기
